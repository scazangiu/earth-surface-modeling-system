# earth-surface-modeling-system
Python project modeling geographic regions and analyzing population and greenhouse gas data, including simulation of future conditions.

**Features**

- Models geographic regions using structured data representations
- Calculates surface area using spherical Earth formulas
- Computes emissions per capita and per square kilometer
- Uses recursion to determine population density and project future conditions

**Technologies Used**

-Python

**Example Functionality**

- Determines the densest region based on population and area
- Simulates future population growth based on terrain type
- Analyzes greenhouse gas emissions across regions

**Status**

- In progress – continuing to expand features and improve simulation accuracy.

- Using recursion to solve complex problems
- Structuring data using Python dataclasses
